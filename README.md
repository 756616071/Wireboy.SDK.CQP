## 先来个自我介绍

这是一个.Net框架下的C#开发的酷Q机器人SDK。嗯...这真是SDK吗？

No，目前还不是，因为你必须使用该项目进行插件开发，还不能单独引用。所以说，目前此项目处于开发的初级阶段。

那么后面会实现单独引用dll就可以开发吗？

如果仅仅只是引用dll就开发，就目前看来是比较困难的，但是后面项目成熟后，我会将SDK功能会逐步独立，然后做一个项目模板，利用项目模板建立自己的机器人插件是完全可行的。
